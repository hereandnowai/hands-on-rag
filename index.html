📋 Step-by-Step Breakdown

Each stage mapped to its key component and purpose.

StepWhat HappensKey ComponentPurpose
⚙️ Load .env secrets dotenv Securely load API keys
⚙️ Initialize LLM & Embeddings ChatGroq + HuggingFaceEmbeddings Set up the brain & encoder
1 Load & chunk the PDF PyPDFLoader + TextSplitter Break document into searchable pieces
2a Create / Load FAISS index FAISS Semantic (meaning-based) search
2b Create BM25 retriever BM25Retriever Keyword (exact-match) search
3 Hybrid search on user query Both + Dedup Best of both search methods
4 LLM generates answer ChatGroq.invoke() Answer grounded in retrieved context
🔑 Why Hybrid Search?

Combining two search strategies produces richer, more accurate retrieval.
